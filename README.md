# Nevada Endemic Plants — Interactive Occurrence Map

Interactive map of Nevada's endemic, near-endemic, and disjunct vascular plant
taxa, combining herbarium records (SEINet/Intermountain Biota) with iNaturalist
observations. Built for field navigation, book-complementary reference, and
Gaia GPS export.

---

## File structure

```
nevada_endemics/
├── index.html                  ← Web app (open this in a browser)
├── fetch_occurrences.py        ← Python data pipeline
├── data/
│   ├── taxa.csv                ← Master taxon list (edit this)
│   ├── occurrences.geojson     ← Generated by fetch script (gitignored if large)
│   └── taxa_metadata.json      ← Generated by fetch script
└── README.md
```

## Quick start

### 1. Edit the taxon list

`data/taxa.csv` contains the 133 endemic taxa transcribed from Tiehm & Nachlinger.
Add near-endemics and disjuncts as you pull them from the book:

```
taxon,genus,endemism_category,esa_status,nv_state_status,book_thematic_category,notes
Lepidium tiehmii,Lepidium,endemic,Endangered,,Edaphic,Boron-tolerant; lithium mine site
Primula nevadensis,Primula,endemic,,,Groundwater Springs,
```

**endemism_category values:** `endemic` | `near-endemic` | `disjunct`

**esa_status values:** `Endangered` | `Threatened` | `Candidate` | (blank)

**nv_state_status values:** `Protected` | (blank)  
  (Nevada tracks rare plants through the Nevada Division of Natural Heritage;
   their tracked species list is at https://ndep.nv.gov/land/natural-heritage)

**book_thematic_category** — suggested values from Tiehm & Nachlinger:
- `Groundwater Springs`
- `Edaphic` (deep loose sand, selenium/boron tolerant soils, etc.)
- `Alpine`
- `Pinyon-Juniper`
- `Sagebrush`
- (leave blank for taxa not singled out in a thematic section)

### 2. Install Python dependencies

```bash
pip install requests pandas tqdm
```

---

### 3. Run the data fetch

**Test run (5 taxa, verify API connectivity):**
```bash
python fetch_occurrences.py --test
```

**Single taxon (useful when you're actively surveying it):**
```bash
python fetch_occurrences.py --taxon "Astragalus lentiginosus var. latus"
```

**Full run:**
```bash
python fetch_occurrences.py
```

**Options:**
```
--inat-rg-only      Only fetch research-grade iNat observations
--skip-seinet       Skip herbarium fetch (iNat only)
--skip-inat         Skip iNat fetch (herbarium only)
```

The script writes:
- `data/occurrences.geojson`  — all occurrence points with metadata
- `data/taxa_metadata.json`   — per-taxon attributes for the info panel

---

### 4. Serve the web app

The app loads GeoJSON via `fetch()`, so it needs a local HTTP server (not
just opening index.html directly — browsers block local file:// fetches).

**Easiest option:**
```bash
cd nevada_endemics
python -m http.server 8080
```
Then open http://localhost:8080 in a browser.

**Or with Node:**
```bash
npx serve .
```

## Map features

**Symbol encoding:**
- **Circle** = herbarium record
- **Diamond** = iNaturalist observation
- **Solid fill** = endemic
- **Open ring** = near-endemic
- **Dashed ring** = disjunct
- **Red** = ESA Listed
- **Orange** = NV State Protected
- **Cycling colors** = individual species (when viewing all taxa)

**Filters:**
- Search / click to isolate a single taxon
- Endemism category chips
- Data source toggle (herbarium / iNat)
- Conservation status chips
- Book thematic category chips
- Genus dropdown
- iNat quality grade chips

**Export:**
Clicking "Export Filtered View" downloads a `.geojson` file of whatever is
currently shown on the map. Import this into Gaia GPS:
- iOS/Android Gaia: Files app → share `.geojson` → Open in Gaia
- Gaia web: Layers panel → Add Layer → Import

The initial data was drawn from the book, "Nevada Vascular Plants: An Annotated Checklist of the State's Botanical Diversity," by Tiehm and Nachlinger:
- near-endemic and disjunct taxon lists
- `book_thematic_category` assignments
- ESA and NV state status flags
- Distribution notes (mountain ranges, counties, elevation bands)

## SEINet API notes

The Symbiota API used here is the v2 occurrence search endpoint at
`https://intermountainbiota.org/portal/api/v2/occurrence/search`.

If you find the API returns 0 results for taxa you know are in the database,
try searching by genus first in the SEINet web interface to confirm spelling,
synonymy, or variety-level resolution. The API accepts partial names.

For taxa with complex synonymy (e.g., Chylismia vs. Camissonia),
add a `synonyms` column to taxa.csv and modify the fetch script to query
each synonym and merge results.

## iNaturalist project (optional complement)

You can create a Collection Project at inaturalist.org scoped to:
- Place: Nevada
- Taxon list: the genera in taxa.csv

This won't display herbarium data, but will auto-collect all NV iNat records
for the relevant taxa and allow the community to add observations.
